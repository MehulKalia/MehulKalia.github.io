Solar Tracker

I made this Solar tracker using Arduino. The solar panels tilt towards light to maximize solar energy captured. Coding was a major part of this project. Basically, the ADC values from the light sensors on either side of the solar panel were compared. The motor turned the solar panel towards whichever side had greater light.

Code

This is the code for my solar tracker project showcased above. The variables "leftPRes" and "rightPRes" represent the ADC values coming from the photoresistors, or light sensors. The higher a photoresistor value is, the brighter the area around the photoresistor is. The difference between the two values is stored in a variable called "difference". At the beginning, the servo rotates the panels towards the middle position. Then, the servo rotates the panels towards the higher photoresistor value side with myServo.write(). The variable servoPos keeps track of the servo's position. Values of the photoresistors are printed using Serial.println() for debugging purposes.

#include <Servo.h> Servo myServo; int leftPRes; //east photoresistor int rightPRes; //west photoresistor int difference; //difference between values int servoPos; //represents servo position int servoMiddle=110; //servo position that makes solar panel flat void setup() { Serial.begin(9600); myServo.attach(9); myServo.write(0); for (int pos = 0; pos <= servoMiddle; pos += 1) { myServo.write(pos); delay(1); } servoPos=servoMiddle; delay(2000); } void loop() { leftPRes = analogRead(A0); rightPRes = analogRead(A1); difference= leftPRes-rightPRes; Serial.println(leftPRes); Serial.println(rightPRes); Serial.println(difference); Serial.println("next iteration"); if (difference>0) { if (servoPos<180); { servoPos+= 5; myServo.write(servoPos); Serial.println(servoPos); } delay(1000); } if (difference<0) { if (servoPos>0) { servoPos-= 5; myServo.write(servoPos); Serial.println(servoPos); } delay(1000); } }

Goldilocks Cup

This project lets you know when your beverage of choice is "just the right" temperature using Arduino, a temperature sensor, LEDs, and coding.

Code

This the my code to detect if a beverage is too hot, too cold, or just the right temperature. The temperature and humidity are measured. If there is enough humidity to accurately measure the beverage temperature, the beverage temperature is compared in if statements to 25 and 60 degrees.

#include <dht.h> dht DHT; #define DHT11_PIN 7 void setup() { Serial.begin(9600); } void loop() { int sensorValue = DHT.read11(DHT11_PIN); int temperature = DHT.temperature; int humidity = DHT.humidity; Serial.print("Temperature = "); Serial.println(temperature); Serial.print("Humidity = "); Serial.println(humidity); if (humidity >= 10) { if (temperature > 60) { digitalWrite(11, HIGH); //Lights Pin 11 LED if too hot digitalWrite(9, LOW); digitalWrite(10, LOW); } else { if (temperature > 25) { digitalWrite(10, HIGH); //Lights Pin 10 LED if just right digitalWrite(9, LOW); digitalWrite(11, LOW); } else { if (temperature <= 25) { digitalWrite(9, HIGH); //Lights Pin 9 LED if too cold digitalWrite(10, LOW); digitalWrite(11, LOW); } else { Serial.println("No temperature could be detected"); } } } } else { Serial.println("Liquid cannot be detected"); //If small humidity liquid needs to be poured more for accurate reading } delay(2000); }

Friday Programming Project: USACO competition

Problem: For a certain number n, how many 13ths of the month fall on each day of the week for n years since January 1, 1900. I used arrays, methods, and loops to solve this problem.

Programming concepts: I programmed using Java and used arrays, methods, and loops to solve this problem.

import java.io.*; import java.util.*; public class friday { public static void main(String[] args) throws IOException { //Setting up input and output BufferedReader inputFile = new BufferedReader(new FileReader("friday.in")); PrintWriter outputFile = new PrintWriter(new BufferedWriter(new FileWriter("friday.out"))); //n represents how many years will we simulate String tempString=inputFile.readLine(); int n=Integer.parseInt(tempString); int whichDayJan=0; int [] daysOfWeek=new int []{0,0,0,0,0,0,0}; int year=1900; int [] tempList; for (int i=0; i<n; i++) { //Checking if year is leap year if (year%400==0||(year%100!=0&&year%4==0)) { tempList=howManyPerYear(true, whichDayJan); whichDayJan=(whichDayJan+1)%7; } else { tempList=howManyPerYear(false, whichDayJan); } year+=1; whichDayJan=(whichDayJan+1)%7; for (int a=0; a<7; a++) { daysOfWeek[a]+=tempList[a]; } } outputFile.println(daysOfWeek[0]+" "+daysOfWeek[1]+" "+daysOfWeek[2]+" "+daysOfWeek[3]+" "+daysOfWeek[4]+" "+daysOfWeek[5]+" "+daysOfWeek[6]); outputFile.close(); } //Method calculates how many friday 13ths in one year per day of week public static int[] howManyPerYear(boolean IsLeapYear, int whichDayJanP) { //daysOfWeek[] will represent how many times 13th of the month lands on each day of week (goes from saturday to friday) int[] daysOfWeekP = new int[]{0, 0, 0, 0, 0, 0, 0}; //daysPerMonth[] and daysPerMonthLeapYear[] represent how many days are in each corresponding month int[] daysPerMonth = new int[]{31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31}; int[] daysPerMonthLeapYear = new int[]{31, 29, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31}; daysOfWeekP[whichDayJanP] += 1; int tempInt = whichDayJanP; int distance; for (int i = 1; i < 12; i++) { if (IsLeapYear==true) { distance = daysPerMonthLeapYear[i - 1]; } else { distance = daysPerMonth[i - 1]; } tempInt += distance; daysOfWeekP[tempInt % 7] += 1; } return (daysOfWeekP); } }

Deer Detection with a Convolutional Neural Network

At Georgia's Governor's Honors Program '56, held at Berry College, there were many deer roaming around campus. Some people may not want deer to enter their yards and eat their vegetation, so I made a program that detects deer using a Convolutional Neural Network and plays an alarm to scare them away.

I trained a Convolutional Neural Network (CNN) through Keras and TensorFlow. The CNN was sent to a Raspberry Pi, which is a small computer. The camera attached to the Pi took a video feed of the environment. The frames of the video feed were modified through an image processing library of Python called OpenCV, and fed into the CNN to detect if there was a deer. If there were deer detected in multiple consecutive frames, an alarm played.

I learned an assortment of concepts through this project, ranging from training a deep learning convolutional neural network to processing images through OpenCV through integrating a deep learning model into hardware through Raspberry Pi.
